Классическая Лазанья

Описание: Традиционное итальянское блюдо, состоящее из слоев теста, мясного соуса болoньезе, соуса бешамель и сыра пармезан.

Для приготовления идеальной лазаньи важно соблюдать баланс соусов. Листы пасты должны пропитаться влагой, чтобы стать нежными, но сохранить структуру.

Секрет шефа: добавьте немного мускатного ореха в соус бешамель для аромата.
